# Evidence Sourcing Guide

> How to find, download, attribute, and add visual evidence to Meridian case files.

---

## How It Works

Each case workspace file (`data/case-XXX-workspace.json`) contains an `investigationTasks` array. These are specific, actionable evidence retrieval missions generated by the AI analyst. Think of it as a scavenger hunt — the AI identifies exactly what evidence exists, where to find it, and what format to save it in. Human investigators go get it.

When you complete a task, update its `status` to `"completed"` and add your name to `completed_by` and the date to `completed_date`.

## License Categories

### Public Domain — US Government Work
Images and videos produced by the US federal government are not subject to copyright. This includes:
- DoD-released UAP videos (FLIR1, GIMBAL, GOFAST)
- Official US Navy photographs of ships, aircraft, equipment
- Congressional testimony documents and hearing transcripts
- FOIA-released documents and imagery
- NOAA weather data, nautical charts, satellite imagery

**Use freely. Credit the source agency.**

### Creative Commons (CC-BY, CC-BY-SA)
- OpenStreetMap tiles (CC-BY-SA — must credit OSM contributors)
- Wikimedia Commons images with CC licenses
- Academic publications under open access

**Use per license terms. Credit as specified.**

### Fair Use — Research/Educational
For copyrighted material used in non-commercial research and analysis:
- News photographs (AP, Reuters, etc.)
- Documentary still frames
- Broadcast video screenshots
- Copyrighted witness footage (e.g., Krzyston video)

**Fair use criteria we meet:** non-commercial research, educational purpose, transformative use (analysis/annotation), limited portion used. **Always credit the original creator and source.**

## How to Complete a Task

1. Find the task in the case workspace JSON (`investigationTasks` array)
2. Follow the `source_url` to locate the evidence
3. Download/extract the specific items described
4. Save to the `save_to` path using the specified `file_names`
5. Update the task:
   ```json
   {
     "status": "completed",
     "completed_by": "Nemo",
     "completed_date": "2026-03-24"
   }
   ```
6. If you find something unexpected or better — add a note and save it anyway

---

## Priority Levels

- **High** — Core evidence that should appear in the workspace UI. These make the case come alive visually.
- **Medium** — Context images that enrich understanding. Maps, reference photos, document scans.
- **Low** — Nice-to-have. Supplementary footage, audio clips, extended document excerpts.
